    About Me

    ผมเป็นนักพัฒนาฟรอนต์เอนด์รุ่นเยาว์ ที่กำลังมองหาบทบาทใหม่ในบริษัทที่น่าตื่นเต้น
    ผมมุ่งเน้นที่การเขียน HTML ที่เข้าถึงได้ง่าย โดยใช้แนวทางปฏิบัติ CSS สมัยใหม่
    และการเขียน JavaScript ที่สะอาดตา เมื่อเขียนโค้ดจาสคริปต์ผมใช้ React เป็นส่วนใหญ่
    แต่ผมสามารถปรับให้เข้ากับเครื่องมือใดก็ได้ที่จำเป็น ผมอาศัยอยู่ในพะเยา แต่ผมมีความสุขที่ได้ทำงานทาง online และมีประสบการณ์ในการทำงานทาง online กับทีม ผมจบการศึกษาระดับประกาศนียบัตรวิชาชีพชั้นสูง สาขา เทคโนโลยีตคอมพิวเตอร์ คอมพิวเตอร์ซอฟต์แวร์ เวลาว่างของผม ผมจะใช้เวลาส่วนใหญ่ในการเรียนรู้สิ่งใหม่ๆ อ่านหนังสือ ฟังเพลง ผมชอบที่จะอยู่ในธรรมชาติไม่ว่าจะไปเดินเล่น วิ่ง เลี้ยงหมา หรือปั่นจักรยาน ผมอยากให้คุณดูงานของผม
